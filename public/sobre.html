Viagem Yūgen - Catálogo de Filmes Studio Ghibli

O Viagem Yūgen é um catálogo digital dedicado aos filmes do renomado Studio Ghibli, um dos estúdios de animação mais influentes e queridos do mundo. O termo japonês "Yūgen" (幽玄) representa uma profunda consciência do universo que evoca sentimentos além das palavras, perfeitamente alinhado com a essência mágica e contemplativa das obras do Studio Ghibli.

Este projeto foi desenvolvido como parte de uma atividade acadêmica para explorar tecnologias web modernas, incluindo HTML5, CSS3, JavaScript, Bootstrap e integração com APIs REST através do JSON Server. O objetivo é oferecer uma experiência interativa e visualmente agradável para os fãs do Studio Ghibli, permitindo explorar, visualizar e gerenciar informações sobre os filmes clássicos do estúdio.

Através deste catálogo, você pode navegar pelos filmes, conhecer suas sinopses, diretores, datas de lançamento e categorias, além de visualizar belas imagens de cada produção. O projeto também inclui funcionalidades de CRUD (Create, Read, Update, Delete) para gerenciar o acervo de filmes de forma dinâmica.
